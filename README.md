# 🔐 パスワードジェネレーター

2026年、毎月1本のWebアプリ制作チャレンジ - 1月の作品

安全で強力なパスワードを簡単に生成できるWebアプリケーションです。

## ✨ 機能

- **カスタマイズ可能なパスワード生成**
  - パスワード長の調整（4〜64文字）
  - 大文字・小文字・数字・記号の選択
  
- **パスワード強度の表示**
  - リアルタイムで強度を評価
  - 視覚的な強度バー表示
  
- **ワンクリックコピー**
  - 生成したパスワードをクリップボードにコピー
  
- **モダンなUI/UX**
  - ダークテーマの美しいデザイン
  - レスポンシブ対応
  - スムーズなアニメーション

## 🚀 使い方

1. `index.html` をブラウザで開く
2. パスワードの長さをスライダーで調整
3. 使用する文字タイプをチェックボックスで選択
4. 「パスワードを生成」ボタンをクリック
5. 生成されたパスワードをコピーボタンでクリップボードにコピー

## 🛠️ 技術スタック

- HTML5
- CSS3（カスタムプロパティ、Flexbox、Grid）
- Vanilla JavaScript（Web Crypto API使用）

## 🔒 セキュリティ

このアプリケーションは**完全にクライアントサイドで動作**し、生成されたパスワードがサーバーに送信されることは一切ありません。

### セキュリティ機能

- ✅ **完全なクライアントサイド処理**: すべてのパスワード生成はブラウザ内で完結
- ✅ **Web Crypto API**: 暗号学的に安全な乱数生成を使用
- ✅ **外部リソースなし**: CDNや外部ライブラリを一切使用していません
- ✅ **HTTPS必須**: Cloudflare Pagesは自動でHTTPSを提供
- ✅ **セキュリティヘッダー**: XSS、クリックジャッキングなどの攻撃を防止
- ✅ **Content Security Policy**: 不正なスクリプトの実行を防止
- ✅ **検索エンジン除外**: 検索エンジンにインデックスされない設定

### プライバシー

- 生成されたパスワードは**サーバーに送信されません**
- パスワードは**ブラウザのメモリ内のみ**に存在します
- ページを閉じるとパスワードは自動的にクリアされます
- **ログやトラッキングは一切行いません**
