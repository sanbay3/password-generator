# 🔐 パスワードジェネレーター

2026年、毎月1本のWebアプリ制作チャレンジ - 1月の作品

安全で強力なパスワードを簡単に生成できるWebアプリケーションです。

## ✨ 機能

- **カスタマイズ可能なパスワード生成**
  - パスワード長の調整（4〜64文字）
  - 大文字・小文字・数字・記号の選択
  
- **パスワード強度の表示**
  - リアルタイムで強度を評価
  - 視覚的な強度バー表示
  
- **ワンクリックコピー**
  - 生成したパスワードをクリップボードにコピー
  
- **モダンなUI/UX**
  - ダークテーマの美しいデザイン
  - レスポンシブ対応
  - スムーズなアニメーション

## 🚀 使い方

1. `index.html` をブラウザで開く
2. パスワードの長さをスライダーで調整
3. 使用する文字タイプをチェックボックスで選択
4. 「パスワードを生成」ボタンをクリック
5. 生成されたパスワードをコピーボタンでクリップボードにコピー

## 🛠️ 技術スタック

- HTML5
- CSS3（カスタムプロパティ、Flexbox、Grid）
- Vanilla JavaScript（Web Crypto API使用）
